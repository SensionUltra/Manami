# Manami
Multipurpose Bot Made by Me and Zac

# Features

* Moderation: 
    - Kick,
    - Ban,
    - Mute,
    - UnMute,
    - Clean,
    - etc

* Economy
    - Balance,
    - Beg,
    - Buy,
    - Items,
    - List

* Per-Guild Settings
    - Custom Prefix,
    - Custom Leave Message,
    - Custom Join Message
